Crypto Analytics 2025

DATA_ENGINEERING + MACHINE_LEARNING + BI

01. El Problema

¿Es posible validar la autenticidad de un movimiento de mercado antes de ejecutar una posición? En los mercados financieros, el "ruido" suele ocultar la rentabilidad. Diseñé un ecosistema para validar estos movimientos mediante ingeniería de precisión y evidencia estadística.

02. Ingeniería de Datos (Arquitectura Medallion)

Utilicé Microsoft Fabric para orquestar un pipeline de datos resiliente:

  • Capa Bronze: Ingesta híbrida mediante Dataflows Gen2 y PySpark.
  • Capa Silver: Limpieza y normalización de esquemas.
  • Capa Gold: Modelado semántico optimizado para BI.

03. BI y Validación

Al cruzar el volumen promedio con la volatilidad, logramos identificar anomalías donde el precio subía sin respaldo de liquidez, permitiendo al inversor evitar "trampas" de mercado.

04. Ciencia de Datos: Predicción

Entrené un modelo de Regresión Lineal para proyectar el precio de cierre mensual. El resultado fue una convergencia estadística con un R² de 0.9863.

En el gráfico superior, se observa cómo la regresión viaja a través de las dimensiones de volatilidad para encontrar el punto de equilibrio.
